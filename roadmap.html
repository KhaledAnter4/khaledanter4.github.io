Offensive Security Roadmap

A high-level path from foundations to bug hunting and vulnerability research.

Designed as a guide, not a strict checklist. Each level emphasizes understanding how systems behave, not just learning tools.

Build Core Understanding

  • Linux usage
  • HTTP understanding

Web Attacker

Think Like an Attacker

  • Trust boundaries
  • Client vs server behavior

Vulnerability Research

Study How Things Break

  • Access control flaws
  • Logic bugs
  • State manipulation

Bug Hunter

Apply at Scale

  • Target selection
  • Chaining vulnerabilities
  • Impact reporting
